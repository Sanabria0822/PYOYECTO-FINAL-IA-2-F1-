#### Objetivo: Entrenar un modelo para reconocer gestos del lenguaje de señas americano (ASL).
- Conjunto de datos: El dataset incluye 68,352 imágenes distribuidas en 40 clases diferentes (gestos de ASL), con una distribución relativamente balanceada entre las clases.
- Modelos: EfficientNetB0, MobileNetV3, CNN.
